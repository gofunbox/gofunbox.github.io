30 个最好的jQuery表单插件

By you

2011-11-08

jQuery 的出现为我们的开发工作带来了极大的便利，而众多的基于jQuery插件的出现就犹如我们在 Firefox 中安装名目繁多的插件一样而乐此不疲，今天带给大家的是精心挑选的这30个最好的jQuery表单插件，希望能提高您的 工作效率，如果您还有更好的，请在次留下您的建议。

Advanced Form Validation


Form Validator的几个特点，如：支持域的分组要求或有条件的验证它是否有效。您也可以验证字符串、数字、日期及邮箱等。您也可以组合多个验证条件。例 如：验证输入的字段必需为数字的且最小长度为5。

jQuery Live Form Validation


帮助您简单地创建表单验证的一款jQuery的插件，高度灵活、有大量的自定义选项。

jquery.validate


jQuery插件帮助您完成输入表单验证中所有单调而重复的工作。您只需要指定哪些表单需要验证，该插件会帮助您处理剩余的部分。

Salid – The Simple jQuery Validator


仅需数行代码，Salid就可以为您完成表单验证。

Just another jQuery validation plugin


这是另外一款具有几个趣味功能的jQuery验证插件

Fvalidate


这是一款Web 2.0 HTML的表单验证插件。

Live Form Validation – jQuery Accessible RIA


近似于富互联网应用程序(RIA)的jQuery框架，基于较为流行的JavaScript框架–jQuery(用户界面工厂容器化)，严格遵循WAI WCAG 2.0 and WAI ARIA的标准以符合Web应用程序的开发。它扩展了标准的HTML表单。支持实时验证，但您还在键入时，它会根据所配置的正则表达式检查每一项。您可以 设置您自己的正则表达式或采用其中的一个预定义模式。

a-tools


跨浏览器的文本选取，是基于jQuery库文件修改而成的插件。

jQuery Doubleselect Plugin


第二个选项框的可填选内容由第一个选项决定，一款jQuery下拉列表联动插件。

After the Deadline – 网页应用中的拼写、语法检查


After the Deadline是一款开源的，提供校对服务的工具。提供网页应用中的拼写和语法检查

AJAX Terminal


jQuery插件，类似于基于浏览器命令console命令的事件行为，执行用户输入的指令AJAX方式提交至服务器，并以文本的方式打印结果集。

ajaxContactForm to simplify te sending of a inquiryform.


jquery.ajaxContactForm 是一款小巧的表单查询发送插件。当表单中必填的值、数值类型的值以及邮箱格式的数据经过了测试，这款插件会将所需的所有值以AJAX的方式提交至服务器。

akeditables


这款插件是基于jEditable创建的，提供了当点击链接时可以创建可编辑区域等。所提供的取代回车键的是保存和取消按钮来保存内容。

akModal: simplest alternative to thickbox


akModal可以简单、快捷地创建模拟对话框或弹出框以供展示一些简单的信息等。

asmSelect – Alternative Select Multiple


替代选择，提供渐进增强功能，选择多个表单元素。

autoclear


使用该款插件可以非常方便地实现在文本域中设置“默认/帮助”字符。

Autocomplete


当用户在文本框中输入字符时，利用搜索和过滤方式自动快速地查找、选择一些值。

autoNumeric


灵活地将动态输入的国际数字格式的字符转变为以特定字符分隔(如千位符)的类型。

bestUpper


BestUpper会将您正在输入的字符实时转换为大写

Checkbox ShiftClick


ShiftClick允许您通过点击复选框来选择，设置一个“锚点”；以shift + 点击方式来选择或取消选择他们之间所有的复选框。

Clearable Text Field


Clearable Text Field 也是jQuery插件，当用户在表单域中输入值的时候，本文框中会显示清除按钮“×”，点击该按钮可以清空文本内容。

ClockPick, a Time Picker


ClockPick是一款时间选择插件，激活用户选择的某个时间值至表单域中。使用独特的弹出层用户界面，ClockPick会花费1/5的时间，通过一 款额外的用户界面选择菜单来输入时间值。

Collapsible Fieldsets


对于那些正在使用Drupal的用户来说，这款jQuery插件可以模拟生成您正在查找字段集。您可以在开启或关闭结构时初始化字段集，缓慢下降。

ComboSelect


如上图所示，可以将选择列表中的多个元素，通过控制选择变换移动到右边的选择列表，反之亦然。如果需要，也可以以字母顺序排列。选择的元素以原始方式提交 表单元素。双击可以将元素切换至另一个选择列表。

Custom HTML form Selectbox


自定义下拉框，取代标准的HTML格式下拉框。允许的标签有disable, multiselect, scrolling,还可以定制标签。

Custom Maxlength


这个插件可以轻松地给自动以的元素应用“maxlength”属性，例如：一个textarea标签。

Form suggestion and validation box in “Twitter style”


类似与Twitter的注册页面，这款插件会重现相同的建议箱样式。用法相当简单。当输入域捕获到焦点，他就会产生通过验证还是验证失败等反馈，显示在 “提示”文本中。

In-field Input Label


jQuery插件inputLabel，使用一个字符串或一个参考标签文本形式，在表单输入域边上插入一个“占位符”。

jListBox


jListbox是jQuery的一个插件，以格式化的方式在下拉列表中显示所应用的格式。

JQF1 JQuery FormOne Form styling


最完整的表格样式插件。

Multiple Select Info


jQuery Multi Select Info可以标识我们都已经选择了哪些是已经被选中的信息。
